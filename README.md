## About TRIVIA+

TRIVIA+ is a RAG-based hallucination detection benchmark that combines
human-verified labels, long context (up to 94K characters — 7–33× longer than
existing benchmarks), sentence-level annotations, and controlled label noise,
satisfying all seven desiderata for hallucination detection evaluation.

The dataset, code, documentation, and usage instructions are available at the
new repository:

**https://github.com/amazon-science/hallucination-benchmark-trivialplus**
